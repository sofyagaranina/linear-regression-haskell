# Linear Regression Project

Курсовой проект по варианту "Алгоритмы машинного обучения, линейная регрессия".

В проекте реализованы:

- обучение линейной регрессии на числовых данных;
- три варианта ADT для регуляризации: `NoRegularization`, `L1`, `L2`;
- оценка качества по метрике `MSE`;
- предсказание для новых объектов;
- сохранение и загрузка модели из текстового файла;
- модульная структура проекта.

## Формат данных

Обучающая и тестовая выборки:

- каждая строка содержит числовые признаки, разделенные пробелами;
- последний столбец это целевое значение.

Пример:

```text
1.0 2.0 8.0
2.0 1.0 9.0
3.0 0.0 10.0
```

Файл для предсказания через сохраненную модель:

- каждая строка содержит только признаки;
- целевого значения в конце нет.

Пример:

```text
9.0 2.0
10.0 4.0
```

## Запуск

Обучение и оценка на тестовой выборке:

```bash
stack run -- train data/train.txt data/test.txt model.txt --lr 0.01 --epochs 4000 --reg l2 --lambda 0.001
```

Предсказание для новых объектов через сохраненную модель:

```bash
stack run -- predict data/predict.txt model.txt
```

Альтернатива через `cabal`:

```bash
cabal run gloss-project-exe -- train data/train.txt data/test.txt model.txt
```

## Что выводит программа

После обучения программа печатает:

- размер обучающей выборки;
- число признаков;
- выбранные гиперпараметры;
- предсказания для тестовых объектов;
- значение `MSE`;
- статус сохранения модели.

## Структура модулей

- `Types` - пользовательские типы и ADT.
- `DataLoader` - чтение и проверка входных файлов.
- `LinearRegression` - обучение и предсказание.
- `Metrics` - вычисление `MSE`.
- `ModelIO` - сохранение и загрузка модели.
- `Report` - формирование итогового отчета.
- `CLI` - разбор аргументов командной строки.
- `Lib` и `Main` - запуск приложения.
